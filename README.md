# 🛒 Customer Shopping Behavior: End-to-End Data Analytics Project

## 📋 Project Overview
This project provides a comprehensive analysis of customer shopping habits to identify key drivers of revenue and customer loyalty. By integrating **Python** for data engineering, **SQL** for deep-dive querying, and **Power BI** for visualization, this project transforms raw transactional data into actionable business intelligence.

## 📊 Dataset
The dataset consists of customer purchase records including demographics (Age, Gender), transaction details (Purchase Amount, Category, Location), and post-purchase metrics (Review Ratings, Subscription Status).
* **Source:** `customer_shopping_behavior.csv`
* **Size:** 3,900+ records

## 🛠 Tools & Technologies
* **Data Processing:** Python (Pandas, NumPy)
* **EDA & Visualization:** Matplotlib, Seaborn
* **Database Management:** MySQL Server
* **Business Intelligence:** Power BI Desktop
* **Presentation:** Gama AI (PPT Generation)

## 🏗 Project Workflow

### 1. Python: Data Cleaning & EDA
* **Initial Audit:** Handled missing values (imputing ratings by category median).
* **Data Standardization:** Formatted column names and standardized currency values.
* **Feature Engineering:** Created `Age Groups` (Young Adult, Adult, Middle-Age, Senior) and mapped `Purchase Frequency` to numerical days.
* **Export:** Cleaned data was prepared for SQL migration.

### 2. SQL: Advanced Analytical Querying
The cleaned data was hosted on a MySQL server to perform complex analysis, including:
* **Customer Segmentation:** Categorized users as "New," "Returning," or "Loyal" based on purchase history.
* **Revenue Analysis:** Calculated total revenue by gender, age group, and shipping type.
* **Product Insights:** Identified top-rated products and those with the highest discount-to-purchase ratio.

### 3. Power BI: Interactive Dashboard
Developed a dynamic dashboard to visualize:
* **Sales Overview:** Total revenue, average order value, and seasonal trends.
* **Demographic Profile:** Breakdown of spending by age and gender.
* **Customer Loyalty:** Distribution of subscription statuses and repeat buyer patterns.

### 4. Reporting & Presentation
* **Project Report:** A detailed summary of the methodology and technical steps.
* **Gama AI PPT:** Created a professional slide deck using Gama to present findings to stakeholders, focusing on business impact and recommendations.

## 📈 Key Results & Insights
* **Loyalty Driver:** Over 60% of revenue is generated by "Loyal" customers (those with $>10$ previous purchases).
* **Demographic Focus:** The "Young Adult" segment shows the highest frequency of high-value purchases.
* **Subscription Opportunity:** Non-subscribers actually have a higher average spend per transaction, indicating a prime segment for conversion marketing.

## 🚀 How to Run
1.  **Environment:** Ensure you have Python 3.x and MySQL Server installed.
2.  **Data Prep:** Run the `Customer_Shop_behaviour.ipynb` notebook to generate the cleaned dataset.
3.  **Database:** Import the `customer_shopping_behavior.sql` file into your MySQL instance.
4.  **Visualization:** Open the `.pbix` file in Power BI Desktop to explore the dashboard.
5.  **Presentation:** View the project summary report and the Gama-generated presentation link (if applicable).
